# API Reference

The full HTTP API is described in the OpenAPI spec and browsable docs.

- OpenAPI (YAML): `/openapi.yaml`
- Redoc UI: `/docs/redoc.html`
- Swagger UI: `/docs/swagger.html`

Use these for up‑to‑date endpoint and schema details. The OpenAPI spec includes:
- Paths, methods, parameters, request/response schemas
- Authentication requirements (Bearer JWT)
- Error envelope `{ error, code }` and common codes
- Pagination (`limit`, `offset`)

See also `docs/README.md` for a quick start and `docs/flows.md` for end‑to‑end examples.
